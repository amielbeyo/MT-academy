How to Use MT academy

Quick Start

Pick a tool from the navigation bar or the homepage. Each one opens in its own tab so you can practice, review answers, and come back later without losing progress.

If you plan to use ChatGPT scoring, drafting, or movement analysis, create your own OpenAI API key before jumping into drills (see API Keys below).

Objections Drill

  1. Select a topic, difficulty, and whether you want a built-in scenario or a new ChatGPT-generated prompt.
  2. Read the short fact pattern and choose the objection you would raise.
  3. Click Check to compare your answer with the expected objection.
  4. Use ChatGPT Argue to practice explaining your reasoning in real time.
  5. Press Score to receive detailed feedback from ChatGPT on the strength of your argument.

Video Coach

  1. Choose the speech type you want to rehearse, then hit Start Recording.
  2. After finishing, click Stop and review using Play or Download.
  3. Select Re-score for updated delivery notes and Show Voice to review the transcript.
  4. Use Movement (ChatGPT) for posture and gesture analysis; if the request is too large the tool falls back to transcript feedback automatically.
  5. Confirm your API configuration anytime with Test ChatGPT.
  6. Final results appear as a rating (Terrible, Bad, Medium, Good, or Amazing) so you instantly know where you landed.

Writing Lab

  1. Pick a document type (opening, closing, cross, etc.) and jot down the facts or themes you want to highlight.
  2. Click Ask ChatGPT to Draft to generate a sample passage tailored to your notes.
  3. Edit the draft directly in the browser and copy it into your preferred editor or team folder.

Rules Library

  1. Search by rule number, keyword, or topic. Results update instantly.
  2. Select a rule to read the full text along with a short explanation.
  3. Use Clear to reset your search or open the Official PDF for the complete rulebook.

Rules Quiz

  1. Choose a mode (standard or lightning), the rule sets you want to review, question count, and difficulty.
  2. Press Start Quiz to begin. Each question appears one at a time—answer, then click Next.
  3. Track your accuracy with the live score display and revisit tricky rules with the review links.

OpenAI API Keys

  1. Log in at platform.openai.com using your own account.
  2. Navigate to Billing → Add payment method, then deposit funds so ChatGPT requests can process.
  3. Open API keys and click Create new secret key.
  4. Copy the key into MT academy's API Key page, choose a model, and hit Save ChatGPT Key.
  5. Return to any tool and click Test ChatGPT (or the equivalent button) to confirm everything works.

Keep your key private. You can remove it at any time with the Remove button on the API Keys page.
